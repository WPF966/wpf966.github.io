CentOS简介

CentOS是免费的、开源的、可以重新分发的开源操作系统 [1] ，CentOS（Community Enterprise Operating System，中文意思是社区企业操作系统）是Linux发行版之一。

CentOS Linux发行版是一个稳定的，可预测的，可管理的和可复现的平台，源于[Red Hat Enterprise Linux](https://baike.baidu.com/item/Red Hat Enterprise Linux/10770503)（RHEL）依照开放源代码（大部分是GPL开源协议 [2] ）规定释出的源码所编译而成。

自2004年3月以来，CentOS Linux一直是社区驱动的开源项目，旨在与RHEL在功能上兼容。

CentOS安装

以下开始在window系统环境下使用虚拟机安装Linux系统

预下载工具

CentOS 7(2009)：https://www.centos.org/download/

VMare Pro下载安装教程：https://www.cnblogs.com/fuzongle/p/12760193.html

虚拟机操作

1、打开VMware选择新建虚拟机

2、典型安装与自定义安装

典型安装：VMware会将主流的配置应用在虚拟机的操作系统上，对于新手来很友好。

自定义安装：自定义安装可以针对性的把一些资源加强，把不需要的资源移除。避免资源的浪费。

这里选择典型安装。

3、选择稍后安装操作系统

4、操作系统选择

选择之后要安装的Linux系统，版本选择CentOS 7 64位

5、虚拟机命名与位置

虚拟机名称可直接命名为CentOS 7 64位

虚拟系统位置在可自行选择电脑硬盘容量充足的分盘位置存储

教程演示选择的是在E盘的位置下，新建一个文件夹用来存储

6、指定磁盘容量

磁盘容量默认20G，虚拟机不会直接占用20G容量，会在虚拟系统中有大量文件的时候才会占用更多容量

默认勾选将虚拟磁盘拆分成多个文件，可以方便虚拟机使用存储设备拷贝复制

7、自定义硬件

修改新CD/DVD(IDE)连接方式，选择使用IOS映像文件，找到下载的CentOS系统文件

网络适配器选择NAT模式

  • NAT模式：就是虚拟机要联网得先通过宿主机才能和外面进行通信
  • 仅主机：虚拟机与宿主机直接连起来

内存与处理器可根据自己需求调整，或者直接默认选项，设置完成后点击关闭。

最后确认，点击完成

安装CentOS

1、开启虚拟机

等待初始化加载，开启虚拟机后出现以下选项

​ Install CentOS Linux 7 安装CentOS 7

​ Test this media & install CentOS 7 测试安装文件并安装CentOS 7

​ Troubleshooting 修复故障

选择第一项，直接安装CentOS 7，回车，等待加载进入下个界面

2、选择语言

这里选择中文简体，美式键盘，点击继续

3、系统安装分区

下拉选择系统、安装位置，可以进行磁盘分区。

选择我要配置分区，点击完成

点击加号，选择挂载点进行分区配置,设置完成好点击完成

/boot 200m

/ 10g

swap 2048m

弹出消息，点击接受更改

设置主机名与网卡信息，打开网络开关即可，设置完成后点击开始安装

安装完成后出现安装配置界面

点击用户设置、ROOT密码，设置密码（密码设置难度弱，点击两次完成就可以设置完成）

等待安装结束进入登录界面

4、修改配置

修改CentOS7的字符集

启用网卡

打开ens33网卡的配置文件/etc/sysconfig/network-scripts/ifcfg-ens33,修改NOBOOT参数为yes。

yum安装基础指令

安装ifconfig命令示例：

根据信息安装net-tools包：

等待加载结束之后就可以使用ifconfig命令了

C++程序的编译

在某些操作系统中，C和C++是同一个编译器，在CentOS中，C的编译器是gcc，C++的编译器是g++。

1、安装g++编译器

用root用户登录服务器，执行以下命令安装或升级gcc-c++编译器。

如果您的CentOS系统没有安装gcc-c++，以上命令就会安装最新版本的gcc-c++，如果已经安装了gcc-c++，就会更新到最新版本的gcc-c++，所以，以上命令不管执行多少次都没有问题。

注释：安装gcc-c++的前提条件是服务器必须可以访问互联网。

2、编译c++程序

编译C程序的命令是gcc，编译C++程序的命令是g++，g++命令和gcc命令的用法相同，把gcc改为g++就可以了，我们在学习C语言时编写的那些示例程序，基本上都可以用g++来编译。
